Hi, this is classic exercises of web scrapping popular websites. Such as LinkedIn.com and Monster.com.

Purpose of this exercises is to learn to fetch or gather data for analysis, which helps in searching job.

Tools used are Python's selenium WebDriver and VScode editor.
 
Website links i.e., https://www.linkedin.in/ and https://www.foundit.in/ are the source of inputs.

Results obtained from this exercises are available in the results folder.
